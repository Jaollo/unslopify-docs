## Three Phases

<CardGrid stagger>
  <Card title="Phase 1: Unslopify" icon="warning">
    18 scripts that remove Windows bloat: Copilot, Recall, telemetry, Edge, OneDrive, AppX bloatware, GPU caches, and more. Launched via console menu or WPF GUI.
  </Card>
  <Card title="Phase 2: Enhance" icon="setting">
    8 standalone scripts that improve your UX: privacy hardening, taskbar cleanup, Explorer tweaks, dark mode, power plan, and more.
  </Card>
  <Card title="Phase 3: Intelligence" icon="document">
    Agent-agnostic reference files and a system scanner. Any AI coding agent can consume these to understand and configure your system.
  </Card>
  <Card title="FOSS Toolkit" icon="star">
    Curated catalog of 20+ open-source tools with winget install commands, config paths, and recommended settings.
  </Card>
</CardGrid>

## Design Principles

- **Safe by default** -- every script follows Scan, Report, Confirm, Execute
- **Reversible where possible** -- registry changes auto-backed up, services can be re-enabled
- **AI-friendly** -- structured for AI-guided execution with Phase 3 system intelligence
